NWS Systems Status — About / Help
Quick open: click Help in the top-right header, press ?, or press Esc to close.

What this app shows

  • An interactive map of NWS systems/sites with status-driven symbology (Up / Down / Degraded / Patching).
  • A right sidebar with:
    • Legend and program legend
    • SID Counter rollups (Type Breakdown)
    • Down Sites table (grouped by program)

Data refresh & sources

  • Point status information is refreshed every 5 minutes.
  • System location information is collected from the NWSLI database, NWS Program records, and field records.
  • Status information is collected from the USOS database, NWS websites, program, and field input.
  • USOS Data is downloaded from the USOS database every 30 minutes.
  • Downloaded USOS data can be viewed at: https://script.google.com/a/macros/noaa.gov/s/AKfycbwW4gvMX66vCSn3YIJf2dPfyUgM2A-8YILqva3sqzdCnEHXe6WyqICawJkkvB12VXosaw/exec

Supplemental program data

  • ASOS — Supplemental ASOS status information is collected every 5 minutes from METARs located at https://tgftp.nws.noaa.gov/data/observations/metar/stations/. If a METAR has not been received from an ASOS in the last 70 minutes the site is flagged as Degraded, after 2 hours the site is flagged as Down. https://tgftp.nws.noaa.gov/data/observations/metar/stations/
  • AWIPS — Supplemental AWIPS status information is collected from the AWIPS program status dashboard. The AWIPS status dashboard is connected to this application, and changes are reflected in the application after the next system refresh. Changes include Service Backup and Patching status changes.
  • Buoys — Supplemental Buoy status information is collected from weather.gov every 15 minutes: https://www.ndbc.noaa.gov/data/latest_obs/latest_obs.txt and https://www.ndbc.noaa.gov/data/realtime2/{station}.txt
  • Facility — Facility data is supplemented through monitoring NCO WAN status information. The NCO dashboard https://www.nco.ncep.noaa.gov/status/dashboard/?view=sites is connected to this application, and changes are reflected in the application after the next system refresh.
  • NWR — Supplemental NWR status information is collected from weather.gov every 30 minutes: https://www.weather.gov/nwr/outages
  • RADAR — Supplemental NEXRAD data is collected from the NEXRAD page on Weather.gov every 15 minutes: https://www.weather.gov/nl2/NEXRADView
  • Upper Air — Supplemental Upper Air status information is collected hourly from weather.gov and Thanks Messages : https://www.weather.gov/upperair/sounding and https://www.nco.ncep.noaa.gov/status/data/thanks/?loc=usa

    • Status meanings

    • Up — normal/operational
    • Down — outage / non-operational
    • Degraded — partially impacted or backup-provided impacts
    • Patching — in security patching window
    • Last Updated (header left) shows the client refresh time.
    • Refresh Timer (header right) shows the countdown/state of the next refresh.

    Filters (left sidebar)

    • Filters container includes:
      • Timed Rotation by Program (optional cycling)
      • Zoom Map to Feature and Show only Down / Degraded sites
      • Advanced Search fields (PID/SID/WFO/City/State/Any text)
      • Regions, Office Type, and Programs selection
      • Reset Filters to return to defaults
    • Most filter and collapse states are persisted (so the app “remembers” your UI choices).

    Map / UI controls

    • MAP/UI includes basemap selection and a UI Settings subsection: UI font scaling, icon size/font scaling, refresh, and Reset Map & Zoom.

    Overlays

    • Overlays are managed inside a single collapsible Overlays container (each has a show toggle + opacity).
    • Included overlays in this build: Radar Base Reflectivity, WWA, WFO Footprint, RFC, CWSU, MCC, Region, and Time Zones.

    Tips

    • Click a site to open the popup (program/site details + outage details when applicable).
    • Use the right sidebar Down Sites grouping chevrons to expand/collapse by program.
    • If something looks stale, try Refresh Data (MAP/UI → UI Settings) and confirm the header’s Last Updated time changes.
    • The rotation function timer can be changed during operation and will adjust immediatly.
    • The rotation function will rotate through filtered points by program.
    • Selecting a point will open a pop-up box with additional information. The rotation will pause until the pop-up is closed.
    • If the Auto Expand checkbox is selected, the Down Sites Table will synchronize with the program rotation.
    • Change your display to full screen (F11) to hide all the browser toolbars.
